# Ghost Access Terminal

Ghost Access Terminal is a fully interactive hacker-style terminal experience built with HTML, CSS, and JavaScript. It combines a Matrix-inspired animated background, a secure login system, command-line interactions, and a mini hacking game to create an immersive cyberpunk-themed web application.

## Features

### Interactive Login System

* Username and password authentication.
* Access denied feedback for invalid credentials.
* Smooth transition from login screen to terminal interface.

### Matrix Background Engine

* Animated Matrix-style falling characters.
* Lightweight canvas-based rendering.
* Responsive to different screen sizes.

### Terminal Command System

* Realistic terminal interface.
* Typing animation for system messages.
* Multiple built-in commands.

Available commands:

* `help` — Display available commands.
* `about` — Display system information.
* `ls` — View virtual files and directories.
* `projects` — View project information.
* `hack` — Launch the hacking simulation mini-game.
* `shadow` — Trigger the Shadow encounter.
* `matrix` — Activate a system event.
* `clear` — Clear the terminal screen.

### Hacking Simulation Mini-Game

* Multi-stage hacking challenge.
* Keyboard reaction gameplay.
* Increasing difficulty:

  * Stage 1: 6 keys
  * Stage 2: 8 keys
  * Stage 3: 10 keys
* 10-second timer per stage.
* Visual warning system:

  * Green = Safe
  * Orange = Warning
  * Red = Danger
  * Panic mode during final seconds.
* Random security alert popups.

### Shadow Protocol

* Hidden narrative event.
* Dynamic red warning messages.
* Creates an immersive system intrusion experience.

### Responsive Design

* Desktop support.
* Tablet support.
* Mobile-friendly layout.
* Adaptive terminal interface.

## Installation

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.

No build tools or external dependencies are required.
